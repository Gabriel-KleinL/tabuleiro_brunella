🎮 Jogo de Tabuleiro

Sistema Multiplayer - Escolha sua tela

🔐 Sala

Use um código de 4 dígitos para criar/entrar em uma sala.

📺

Visualizador

Mostra o tabuleiro para todos verem. Controle o fluxo do jogo e gerencie os turnos.

 🎯

Jogador

Para cada equipe fazer suas jogadas. Selecione sua equipe e jogue seu turno.

📋 Como usar:

  1. Visualizador: Abra em uma tela grande para todos verem
  2. Jogador: Abra uma tela para cada equipe participante
  3. Todas as telas se sincronizam automaticamente!
💡 Dica: Para melhor experiência, abra cada tela em uma aba separada do navegador.
